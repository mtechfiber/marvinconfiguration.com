MCFG STANDALONE LANDING PAGE

1. Upload index.html to your hosting or public_html folder.
2. Open index.html in any text editor.
3. Search for: href="https://mcfgcloud.com/
4. Replace each URL with your preferred direct link.

Important direct links currently included:
- VPN Login Portal: https://mcfgcloud.com/login
- Contact buttons: https://www.facebook.com/marvinqmarvida/

The page is self-contained: CSS and JavaScript are already inside index.html.
No npm installation, database, API, or build step is required.

LOGO FILE
- Keep logo-mcfg-transparent.png in the same folder as index.html.
- The logo is used in both the header and footer.

LIGHT / DARK MODE
- The sun/moon button in the top navigation switches themes.
- The visitor's choice is saved automatically in the browser.
- On the first visit, the page starts in light mode; visitors can switch to dark mode.
